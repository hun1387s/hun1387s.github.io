``` bash
git init
```

터미널에서 어느 경로에 있든, 그 경로를 새 저장소로 초기화 한다.

터미널에 있던 경로는 저장소의 홈 경로가 된다.

프로젝트 당 한번만 실행된다.



``` bash
git status
```

저장소의 상태를 보고해주는 명령어.

(현재 위치한 저장소에 대한 정보)
