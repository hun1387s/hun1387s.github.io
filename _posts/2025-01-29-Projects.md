# Project

## Smooth Node 제작 (Maya) 

### 기간

​	2023.10 - 2024.01 

### 프로젝트 개요

- 기존 Maya 및 타 소프트웨어를 병행하여 수행했던 **Smooth 작업을 단일 Maya 노드로 개발**하여 작업 과정 간소화 및 효율성을 극대화.
- 클로스 시뮬레이션을 위한 **Mesh Smooth Node**를 내부적으로 구현하여 Maya에서의 시뮬레이션 정확도 및 성능을 향상.

### **문제 해결 및 성과**

✅ **프로세스 최적화**: 타 프로그램을 사용할 필요 없이 **Maya 내부에서 Smooth 처리가 가능**하도록 개선. 

✅ **시뮬레이션 향상**: 개발된 Smooth Node를 통해 **클로스 시뮬레이션의 안정성과 정밀도 증가**. 

✅ **비용 절감**: 타 소프트웨어 사용 시 발생하는 **라이선스 비용 절감 및 교육 부담 완화**. 

✅ **협업 효율 향상**: 모든 작업을 Maya 내에서 처리할 수 있도록 하여 **팀원 간 작업 속도 증가 및 커뮤니케이션 효율성 향상**.

### **주요 역할**

- 프로젝트 설계 및 개발 수행.
- 기술적 검토 및 팀원 간 협업 조율.
- Maya API를 활용한 **Smooth Node 알고리즘 개발 및 성능 개선**.

### **사용 기술**

- **Autodesk Maya** (Maya API, Node 개발)
- **Python** (Maya API 활용)
- **3D Mesh 처리 기술**
- **Box Blur 알고리즘** 활용

## 프로젝트: 사내 오토리깅 툴 개발 (Maya)

### 기간

​	2023.3 - 2023.12 

### **프로젝트 개요**

- 다양한 캐릭터에 대해 **맞춤형 자동 리깅(Auto Rigging) 시스템을 개발**, 리깅 시간 단축 및 품질 유지.
- 리깅된 캐릭터들이 동일한 구조를 가지도록 설계하여 **애니메이션 키 호환성을 향상**시킴.

### **문제 해결 및 성과**

✅ **리깅 자동화**: 반복적인 리깅 작업을 자동화하여 **작업 시간 절감 및 생산성 향상**. 

✅ **애니메이션 호환성 개선**: 캐릭터 간 **동일한 리깅 구조 유지**로 인해 애니메이션 재사용성 증가. 

✅ **작업 효율성 증대**: 애니메이터들이 보다 **빠르고 일관된 리깅 환경에서 작업할 수 있도록 지원**. 

✅ **사용자 친화적 인터페이스**: **직관적인 UI/UX** 설계로 사내 사용률 증가 및 전사적 도입 촉진.

### **주요 역할**

- 애니메이션 팀과의 협업을 통해 요구사항 분석 및 기능 설계.
- Auto Rigging 기능 개발 및 Maya API 기반 구현.
- 팔, 다리 등의 주요 리깅 시스템 설계 및 테스트.

### **사용 기술**

- **Autodesk Maya** (Maya API 활용)
- **Python** (Maya API를 활용한 리깅 자동화 스크립트 개발)
- **Auto Stretch, Soft-IK, Round-IK, Bending Mechanics** 구현

## 프로젝트: 언리얼 Sequencer를 활용한 Visibility Animation 적용

### 기간

​	2022.9 - 2023.1

### **프로젝트 개요**

- **언리얼 엔진(UE) Sequencer를 활용하여 Mesh Visibility Animation 시스템을 구축**, 블루프린트를 통해 실시간 제어 가능하도록 개발.
- 캐릭터 BP(블루프린트)에 다양한 Skeletal Mesh를 설정하여, 애니메이션 중 **Mesh Visibility를 동적으로 조정 가능**.

### **문제 해결 및 성과**

✅ **시스템 통합**: 언리얼 엔진 내에서 **캐릭터의 Mesh Visibility를 실시간 제어하는 솔루션을 제안 및 구현**. 

✅ **작업 효율성 향상**: 기존보다 **더 직관적이고 빠른 Visibility 제어 방법을 제공**하여 애니메이션 팀의 작업 부담 경감. 

✅ **사용자 가이드 제공**: 시스템 활용법을 문서화하여 **팀원들이 쉽게 접근 및 활용할 수 있도록 지원**. 

✅ **혁신적 접근 방식**: 기존 복잡한 Visibility 관리 방법을 개선하여 **시간 절약 및 비용 절감**.

### **주요 역할**

- Visibility Animation 기능 설계 및 기술 검토.
- 언리얼 엔진 내 **블루프린트 기반 솔루션 구현 및 테스트**.
- 내부 사용자를 위한 문서화 및 가이드 작성.

### **사용 기술**

- **Unreal Engine** (Sequencer 활용)
- **Blueprints** (Visibility Animation 로직 구현)
- **Skeletal Mesh Visibility 컨트롤**
