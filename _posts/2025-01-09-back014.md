Git을 사용하기에 앞서 내 정보를 등록해야한다.

 

먼저 이름 등록은 아래처럼 등록한다.

``` bash
git config --global user.name <YourName>
```





이후 이메일 등록은 아래처럼 등록한다.

``` bash
git config --global user.emal <Your@Email.com>
```
